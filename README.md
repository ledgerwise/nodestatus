# nodestatus
Monitoring tool for eosio networks

## Usage 
```bash
usage: nodestatus.py [-h] [-v] [-d] [-l LOG_FILE]

optional arguments:
  -h, --help            show this help message and exit
  -v, --verbose         Print logged info to screen
  -d, --debug           Print debug info
  -l LOG_FILE, --log_file LOG_FILE
                        Log file
```
